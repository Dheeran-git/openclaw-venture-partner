# OpenClaw Venture Partner — Dashboard UI Kit

A pixel-fidelity recreation of the Venture Partner web dashboard. Three core surfaces:

1. **Lead inbox** — sortable table with score badges, layer/source tags, status pills.
2. **Lead detail** — right-hand pane with pitch draft, proof-of-value preview, approval bar.
3. **Client memory** — markdown notes on each won client, used by the agent to personalize follow-ups.

Open `index.html` for the full assembled view. Click any lead to open its detail pane; click "Approve & send" to send a pitch (mocked).

## Files
- `index.html` — assembled dashboard
- `Sidebar.jsx` · `Topbar.jsx` · `LeadTable.jsx` · `LeadDetail.jsx` · `PitchCard.jsx` · `ActivityRail.jsx` · `StatCards.jsx`
- `data.js` — fake lead/pitch/client fixtures
